## finch rm

Remove one or more containers

```
finch rm [flags] CONTAINER [CONTAINER, ...]
```

### Options

```
  -f, --force     Force the removal of a running|paused|unknown container (uses SIGKILL)
  -h, --help      help for rm
  -v, --volumes   Remove volumes associated with the container
```
